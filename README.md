# ipLogger
👀 traefik reverse proxy real ip test  

## purpose
A study on how services located behind multiple reverse proxies log real client IPs.  

여러 리버스 프록시 뒤에 위치한 서비스가 실제 클라이언트 IP를 기록하는 방법에 대한 연구
